# Мои проекты

## Мобильные приложения

- [Mopup (App Store)](https://apps.apple.com/ru/app/mopup/id6756664260) — Мобильное приложение клинингового сервиса Mopup. Заказ профессиональной уборки в пару кликов: выбор услуги, назначение времени, управление заказами.

- [Cleaner PRO (App Store)](https://apps.apple.com/ru/app/cleaner-pro/id6737475988) — Приложение для исполнителей сервиса Mopup. Коммуникация с базой заявок, управление заказами на стороне клинеров.

- [Cleaner PRO (RuStore)](https://www.rustore.ru/catalog/app/com.coderok.cleanerpro) — Android-версия приложения Cleaner PRO для исполнителей клинингового сервиса Mopup.

- [FinFlow PRO (App Store)](https://apps.apple.com/us/app/finflow-pro/id6752355013) | [GitHub](https://github.com/Coderok-ru/fin_flow_pro) — Мобильное приложение для управления личными финансами на Flutter + GetX. Учёт доходов/расходов, бюджетирование, аналитика и календарное планирование.

- [Cryptor](https://github.com/Coderok-ru/cryptor) — Кроссплатформенное приложение (macOS, Windows) для шифрования CSV-файлов и текста по протоколу AES. Безопасная передача зашифрованных данных через открытые каналы связи.

- [AI Assistent (App Store)](https://apps.apple.com/us/app/ai-assistent/id6748020233) | [GitHub](https://github.com/Coderok-ru/ai_assistent) — Мобильное приложение — карманный AI-ассистент.

## Веб-проекты

- [Artfabric](https://artfabric.ru) — Интернет-магазин дизайнерской мебели, светильников и декора. Каталог с категориями, карточки товаров, корзина.

- [Artfabric CMS](https://cms.artfabric.ru) — Система управления контентом для интернет-магазина Artfabric. Административная панель для управления каталогом, заказами и контентом.


- [Ivinsight](https://ivinsight.ru) — Платформа онлайн-школы осознанного питания Валерии Любиченко. Курсы, калькуляторы здоровья, тесты, рецепты, личный кабинет с доступом к урокам.

- [Mopup](https://mopup.ru) — Веб-сайт клинингового сервиса Mopup. Профессиональная уборка в Санкт-Петербурге.

- [Slipstream Karting](https://slipstream-karting.ru) — Сайт картинг-клуба Slipstream в Санкт-Петербурге. Бронирование заездов, прайс-лист, галерея, школа картинга.
